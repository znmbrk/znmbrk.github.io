Animating Maths

Using JS to showcase the beauty of maths!

Project Explanation

Mathematics has a certain profound order, beauty and simplicity to it. My aim is to express this side of a rather daunting subject in a way that highlights how awe-inspiring it can be.

I produce my animations using Java Script's p5 Library. This is a library for creative coding, with a focus on making coding accessible and inclusive for artists and designers! p5.js has a full set of drawing functionality.

Project 1: Fourier Series!

A Fourier series is a sum that represents a periodic function as a sum of sine and cosine waves.
